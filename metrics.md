Context Scattering (Dispersão)	Quantos arquivos diferentes o agente altera para uma única user story. Quanto maior a dispersão, maior o risco de alucinação.


Architectural Violation Rate (AVR)	A frequência com que o agente quebra as regras da CMA (ex: injetar uma query SQL diretamente em um handler).


Token-to-Logic Ratio	Quantos tokens de "explicação" ou "contexto" o agente consome antes de chegar à implementação correta.


Dependency Depth (Profundidade)	O número de saltos em imports que o agente precisa seguir para mapear um Domain Model até o banco de dados.
